enable-hyperv-sandbox

Enable hyper-v and sandbox even if you are on windows home!

Prerequisites

  • Windows 10 or newer
  • Internet connection required

Installing

You can get the stable releases here.
Or you can build using visual studio (see below)

Build

Prerequisites

  • Visual studio 2019 or newer
  • Windows 10 or newer

Visual studio

  • Install git if you have not.
  • Clone the repository:
git clone https://gitlab.com/wcyat/enable-hyperv-sandbox.git
  • open ehs.sln using visual studio
  • Build
